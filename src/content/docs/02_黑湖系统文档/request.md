1. 报工扫码扫入重量，系统校验（由于包装方式多变，先立项不实行）
2. 包覆打胶的材料（面料）  要在MES 有核对 正确性的过程
    黑湖下发任务，产出物料必须维护物料版本或者物料清单默认版本，产线收到任务执行投料，系统就能识别所用物料是否准确（蒋颜+盛佳瑶）
		
2. 作业前后完成，系统打印标贴（匹配系统）贴到单一产品上待组装。
    从生产工单预生成标签打印，提供模板后配置（刘晓刚+戚海斌）

3. 组装前扫码匹配单一整椅椅子上。（扫码显示正确的信息用显示器表示出来）
    同1，增加电脑、显示器、扫码枪
	
4. 完成组装的椅子（记录完成数量和未完成数量，进度条显示到总屏幕上）
    生产工作台可以查看
    ![](./assets/Pasted%20image%2020260207141317.png)

5. 完成检验SRP 检验各项。

    提供检验方案后配置，一般根据产品分类

6. 增加IFID 门禁统计（后续对接ERP完成数量交接）
    建议1-5完成后建设
