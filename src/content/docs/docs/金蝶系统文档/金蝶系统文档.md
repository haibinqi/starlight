## 二、采购管理

### 物料采购信息

**影响计划的采购属性配置主要包含**
- 固定提前期（采购周期）
- 默认供应商

### 物料超收控制

**物料收货控制**
物料启用“控制收货数量”，维护“收货超收比率”后新订单开始生效，原订单要做订单变更

### 采购订单

**交货日期变更**
采购订单开放交货日期字段审核后可修改。采购在接收供应商或者计划部门提出修改交货日期的请求后及时修改。修改方法：直接打开采购订单明细，修改数据。

## 三、库存管理

### 物料库存信息

**关键属性维护**
- 默认仓库仓位（对应采购入库仓库仓位）
- 启用安全库存、安全库存数量
- 启用批号管理
- 启用序列号管理
- 发料仓位仓位
- 汇报入库仓库仓位

### 盘点方案

**盘点表**
盘点表按仓库分表，每个仓库导出一个盘点表。解耦。

## 四、计划管理

### 物料计划信息
**影响计划的关键字段**
- 批量政策、批量规则，当前设置按周合并计划特例SO号不一样的情况下不允许合并
- 最小订货量

### 计划订单

**计划订单提交控制**
控制：计划订单控制没有BOM编号的计划订单不允许提交和保存

### 计划运算向导

**已下销售订单后修改bom，执行计算运算向导报错**
处理方式：
1. 销售订单行选中后点击“预留查询”
2. 在预留查询页面点击“预留解除”

**预留计划订单**
20260126开始重复运行计划运算向导考虑已有计划订单、系统不会重复生成计划订单

## 五、财务管理

### 汇率
每个月初财务维护美元、欧元汇率，汇率表为固定汇率

## 六、工程制造数据管理

### 物料

### 物料新增流程
物料新增流程，工程部新增通知采购、计划、仓库、等部门维护关键属性后审核。特殊情况可以先审核但是必须通知（暂时不启用流程，流程相对比较耗时初期不建议）

| 属性   | 核心字段                                | 部门责任人       |
| ---- | ----------------------------------- | ----------- |
| 基础属性 | 物料编码、名称、规格型号、图号                     | 蒋颜          |
| 采购属性 | 默认供应商、固定提前期                         | 张晓燕 and 沈开林 |
| 计划   | 提前期、批量规则（生产订单合并）、最小订货量、安全库存、批号序列号管理 | 计划/王子强      |
| 仓库   | 默认仓库仓位（来料）、发料仓库仓位、入库仓库仓位            | 范燕娟         |
| 质量   | 来料检验、生产检验、发货检验、检验方案                 | 侯水辉         |
### 虚拟件

**虚拟件设置**
虚拟件PH必须设置为生产领料，系统自动跳层到下一级

### 物料版本管理
- 工程部新建升版后的物料（控制规则：只有功能、尺寸、配合关系变更才新建新料）
- 仓库做库存形态转换，把原有库存转移到升版后的物料
- 原料号停用

### BOM维护

**BOM变更联动用料清单更新**
工程部门变更bom后必须更新生产工单的用料清单

## 七、委外管理

###  流程

**线下流程**
委外工单执行单模板，本质包含：委外任务通知+用料确认，委外工单执行单格式设置如下

| 表头信息·                        | 用料清单                               |
| ---------------------------- | ---------------------------------- |
| 委外工单、委外供应商、产品编码/名称、委外数量、计划交期 | 物料编码、物料名称、计划用量、本次实发、仓库签名、委外签名、发料日期 |
### 直接调拨单

**委外调拨库存到供应商仓操作步骤**
1. 单据类型选“产品委外直接调拨单”
2. 调入仓库选供应商委外仓
3. 调入保管着选对应供应商

## 八、生产管理

### 生产工单

**工单没有bom**
生产工单没有没有BOM编码的工单，说明下工单时没有BOM待工程补全BOM后从工单行指定BOM，操作方式：选中工单行点击“指定BOM”按钮
